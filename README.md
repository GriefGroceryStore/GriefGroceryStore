# 解忧杂货店(GriefGroceryStore)
浙江大学智能终端软件开发小组作业。这是一款基于Android Studio开发的匿名倾诉交友软件，以《解忧杂货店》为参考，用户可以通过这个APP安全、私密地实现与陌生人的交谈，达到释放内心情感的效果。

FROM ZJU. An Android Application for people to sooth their grief.
